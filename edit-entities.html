Advanced Mutable Behaviors JSON

This textarea lets you define custom or complex behaviors that extend or override the structured fields. Use valid JSON object format. Keys are events/actions, values are objects with details.

Basic Syntax:

{
  "onEvent": {
    "action": "value",
    "anotherAction": {"key": "detail"}
  }
}

Common Events:

  • "onDestroy": Triggers when entity is destroyed/removed
  • "onInteract": Triggers on player tap
  • "onTimer": Time-based (e.g., growth)
  • "onCondition": Conditional checks

Common Actions:

  • "drop": Array of items to drop (overrides structured drop field)
  • "transform": Object with "to" (new entity) and optional "map" (switch map)
  • "dialogTree": Array of dialog nodes (overrides structured dialog)
  • "giveItem": String of item to add to inventory (overrides structured give)
  • "spawn": String of entity to spawn
  • "every": String like "1h" for timers
  • "if": String condition like "zoneHealth > 80"

Dialog Tree Examples – Ready to Copy-Paste

1. Simple linear dialog

[
  {"message": "The ancient tree whispers..."},
  {"message": "Protect the island, young gardener."},
  {"message": "Take this seed as thanks."}
]

2. One question with two endings

[
  {
    "message": "Do you want to help restore the beach?",
    "choices": [
      {"text": "Yes, show me how!", "next": 1},
      {"text": "Not now", "next": 2}
    ]
  },
  {"message": "Great! Here's a watering can."},
  {"message": "Come back when you're ready."}
]

3. Branching with item reward

[
  {
    "message": "The forest is choked with vines...",
    "choices": [
      {"text": "I'll clear them!", "next": 1, "giveItem": "shovel"},
      {"text": "I can't right now", "next": 2}
    ]
  },
  {"message": "Thank you! Take this shovel and start cutting."},
  {"message": "The vines grow stronger each day... good luck."}
]

4. Multi-step conversation

[
  {
    "message": "Who goes there?",
    "choices": [
      {"text": "Just a gardener", "next": 1},
      {"text": "A warrior!", "next": 3},
      {"text": "Leave me alone", "next": 2}
    ]
  },
  {"message": "Ah, a friend of the earth. Take this leaf."},
  {"message": "As you wish..."},
  {
    "message": "A warrior? Prove it — defeat the thistle!",
    "choices": [
      {"text": "Challenge accepted!", "next": 4},
      {"text": "Maybe later", "next": 2}
    ]
  },
  {"message": "Return when the thistle is gone."}
]

5. Complex conditional NPC dialog

[
  {
    "message": "Have you cleared the beach yet?",
    "choices": [
      {
        "text": "Yes!",
        "condition": "zoneHealth.beach > 70",
        "next": 1,
        "giveItem": "rare-seed"
      },
      {
        "text": "Not yet",
        "next": 2
      }
    ]
  },
  {"message": "Wonderful! Here's a rare seed for your efforts."},
  {"message": "Come back when the beach is healthier."}
]

Tips for dialog trees:

  • "message": What is said
  • "choices": Array of player options
  • "text": What player sees
  • "next": Index of next node (0 = first message)
  • Optional: "giveItem", "condition", etc. inside choices
  • No choices = linear dialog (click to continue)

Validation & Errors:

  • Invalid JSON in advanced area is ignored (warning shown)
  • Use jsonlint.com if syntax is tricky
  • Game must implement support for new keys (e.g., "condition")
